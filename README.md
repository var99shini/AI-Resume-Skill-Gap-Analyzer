# AI Resume Skill Gap Analyzer

This project uses Artificial Intelligence and Natural Language Processing (NLP) to analyze resumes and compare them with job descriptions.

## Features
- Resume and Job Description comparison
- Skill gap identification
- Match percentage calculation

## Technologies Used
- Python
- NLP
- TF-IDF
- Cosine Similarity
- Jupyter Notebook

## How to Run
1. Open the Jupyter Notebook
2. Run all cells
3. Modify resume and job description text to test

## Output
- Resume match percentage
- Missing skills list
